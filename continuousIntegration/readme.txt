Steps to run YUITest

Pre-required
 - downloaded Python 2.7.1
 - VMPlayer with simulator
 - Webwork Tablet SDK

 
To Run Tests 
1/ Changed the ip and host @pythonServerIP@:@pythonServerPort@ for
  - server.py 
  - yui/YUIHarness.js 

2/ Build you App or use the YUITest
  - change the @pythonServerIP@:@pythonServerPort in config.xml
  - change @buildid@ to your build id number

3/ Run server.py

4/ compile and deploy your app to your simulator

5/ Result file locate where your server.py
